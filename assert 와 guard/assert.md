# assert
* 조건이 틀리면 프로그램을 일부러 멈추게 해서 버그를 잡는 코드



**assert 함수는 디버깅 모드(코드를 실행하면서 오류(버그)를 찾고 확인하는 상태)에서만 작동함**(중요!)

*예시*
```swift
let score = 120

assert(score <= 100, "점수는 100 이하이어야 합니다")

print("정상 실행")
// score가 120이라서 조건 false이 되서 결과가 출력이 안된다
```
**특징**
1. 배포하는 앱에서는 제외됨 
2. 주로 디버깅 중 조건의 검증을 위하여 사용함
