# 프로퍼티
**클래스나 구조체 안에 들어있는 값**

* 객체가 가지고 있는 변수

## 프로퍼티의 종류
- 저장 프로퍼티
    - 값을 직접 저장하는 프로퍼티
```swift
struct Person {
    var name: String = "시우"
    let age: Int = 17
}
```
- 연산 프로퍼티
    - 값을 저장 안 하고 계산해서 반환
```swift
struct Rectangle {
    var width: Int
    var height: Int
    
    var area: Int {
        return width * height
    }
}
```
- 인스턴스 프로퍼티
    - 클래스/구조체로 만든 객체(인스턴스)에 속한 프로퍼티
```swift
struct Person {
    var name: String
    var age: Int
}

var person1 = Person(name: "시우", age: 17)
var person2 = Person(name: "짱구", age: 5)

print(p1.name) // 시우
print(p2.name) // 짱구
```
- 타입 프로퍼티
    - 클래스/구조체 자체에 속한 프로퍼티
```swift
struct Math {
    static let p = 3.14
}
```
### 프로퍼티는 구조체, 클래스, 열거형 내부에 구현할 수 있다

다만 열거형 내부에는 연산 프로퍼티만 구현할 수 있고

연산프로퍼티는 var로만 선언할 수있다


## 지역변수, 전역변수)에도 사용가능
저장 프로퍼티와 연산 프로퍼티의 기능은
함수, 메서드, 클로저, 타입 등의 외부에 위치한
지역/전역 변수에도 모두 사용 가능함
```swift
var a: Int = 100
var b: Int = 200

var sum: Int {
    return a + b
}

print(sum) // 300
```
