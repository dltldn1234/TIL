# 조건문
**If -else - 조건에 따라 다른 코드를 실행하는 문법**
* 만약 ~ 이면 이거 하고, 아니면 저거 해라 라는 뜻
```swift
if 조건 {
    실행
} else {
    실행
}
```
*사용예시*
```swift
let age = 17

if age >= 19 {
    print("성인")
} else {
    print("미성년자")
}
```

**Switch - 값에 따라 여러 경우 중 하나를 선택해서 실행하는 문법**
* 이 값이면 이거, 저 값이면 저거 실행
```swift
switch 값 {
case 조건:
    실행
default:
    실행
}
```
*사용예시*
```swift
let num = 2

switch num {
case 1:
    print("1")
case 2:
    print("2")
default:
    print("다른 값")
}
```
**범위 연산자를 활용하면 더욱 쉽고 유용하다**
```swift
let score = 85

switch score {
case 90...100:
    print("A")
case 80..<90:
    print("B")
default:
    print("C")
}
```
