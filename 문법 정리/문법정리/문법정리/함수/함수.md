# 함수
* 코드를 묶어서 재사용하는 것


## 함수선언의 기본형태
```swift
func 함수이름( 매개변수1이름: 매개변수1타입, 매개변수2이름: 매개변수2타입 …) -> 반환타입 { 
    함수 구현부  
    return 반환값 
}
```
```swift
func 함수이름() {
    실행할 코드
}
```
**반환값이 없을때**
```swift
// 반환값이 없으면 void
// 아예 생략도 가능
func 함수이름( 매개변수1이름: 매개변수1타입, 매개변수2이름: 매개변수2타입 …) -> void { 
    함수 구현부  
    return 
}
```
**매개변수가 없는 함수**
```swift
func 함수이름() -> 반환타입 {
    함수구현부
    return 반환값
}
```

**매개변수와 반환값 둘다 없을때**
```swift
func 함수이름() -> void {
    함수 구현부
    return
}
```
# 고급

**기본값을 갖는 매개변수는 매개변수 목록 중에 뒤쪽에 위치하는 것이 좋다**
```swift
func greet(name: String = "시우") {
    print("안녕 \(name)")
}
```
**매개변수 기본값을 가지는 매개변수는 생략가능**

# 전달인자 레이블
* 전달인자 레이블은 함수를 호출할 때 매개변수의 역할을 좀 더 명확하게 하거나 함수 사용자의 입장에서 표현하고자 할 때 사용한다
```swift
func 함수이름(전달인자 레이블, 매개변수1이름: 매개변수 1타입, 전달인자레이블 매개변수2이름: 매개변수2타입 ...) -> 반환타입{
    함수 구현부
    return
}
```
* 함수 내부에서 전달인자를 사용할 때에는 매개변수 이름을 사용한다
```swift
func greeting(to friend: String, from me: String) {
    print("Hello \(friend)! I'm \(me)")
```

* 함수를 호출할 때에는 전달인자 레이블을 사용해야 한다

# 가변매개변수
* 전달 받을 값의 개수를 알기 어려울 때 사용할 수 있다
* 가변 매개변수는 함수당 하나만 가질 수 있음
```swift
func 함수이름(매개변수1이름: 매개변수1타입, 전달인자 레이블 매개변수2이름: 매개변수2타입...)-> 반환타입 {
    함수 구현부 
    return
}
```
# 데이터 타입으로서의 함수
**함수의 타입 표현**
* 반환타입 생략 불가능
```swift
// (매개변수1타입, 매개변수2타입 ...) -> 반환타입
```
```swift
var someFunction: (String, String) -> Void = greeting(to:from:)
someFunction("eric", "siwoo") // Hello eric! I'm siwoo
```
**타입이 다른 함수는 할당 불가**
```swift
//someFunction = sayHelloToFriends(me: friends:)

func runAnother(function: (String, String)-> Void) {
    function("jenny", "mike")
}
```
