# 상속
**기존 클래스의 기능을 물려받아서 새로운 클래스를 만드는 것**
* 스위프트의 상속은 클래스, 프로토콜 등에서 가능함

열거형, 구조체는 상속이 불가능함

* 스위프트는 다중상속을 지원 안함( 단일 상속이여서 )

## 클래스의 상속과 재정의
```swift
class 이름: 상속받을 클래스 이름 {
    /* 구현부 */
}
```
*예시*
```swift
class Person {
    var name: String = ""
    
    func selfIntroduce() {
        print("저는 \(name)입니다")
    }
}
```

## final 키워드를 사용하여 재정의를 방지할 수 있음

## 타입 메서드(인스턴스가 아니라 “타입 자체”에서 호출하는 메서드)

### 재정의 불가 타입 메서드 - static

### 재정의 가능 타입 메서드 - class

* 재정의 가능한 class 메서드라도 final 키워드를 사용하면 재정의 할 수 없다

* 메서드 앞의 ‘static’과 ‘final class’ 는 똑같은 역할을 함

* static을 사용한 타입 메서드는 재정의 할 수 없고
final 키워드를 사용한 메서드, 프로퍼티는 재정의 할 수 없음
