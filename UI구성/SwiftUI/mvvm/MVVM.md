# MVVM 
* 아키텍쳐 패턴 중 한개의 종류

UI = 보여주기식    <-    controll   <-   logic = 기능(비교)   <- mode

한 폴더안에 ui랑 logic을 같이 넣을 수 있음
근데 안 넣는다 파일이 삭제 되면 유아이랑 로직이 다 날라가서

# 로직
* 만약 서버에서 로그인 기능을 줬어 서버가 알아먹을수 있는 언어로 스위프트 언어로 주지 않고 준것을 변환한다 스위프트가 알아먹을수 있도록 변환 하는것 그것이 계산 그것이 로직파일 안에서 일어 나는 것이다

한파일에 기능이랑 ui를 넣지 않는다

각 기능 들을 모두다 따로 따로 만들고 그것들을 연결 시키는것이 컨트롤

내가 장님이고 오물결이 말을 못하면 서로 도와줘야 되는데
둘이 멀리 떨어져 있으면 못 도와주니까 가까이 같이 있는게 컨트롤!
