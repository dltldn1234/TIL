# 문제
준하는 사이트에 회원가입을 하다가 joonas라는 아이디가 이미 존재하는 것을 보고 놀랐다. 준하는 놀람을 ??!로 표현한다. 준하가 가입하려고 하는 사이트에 이미 존재하는 아이디가 주어졌을 때, 놀람을 표현하는 프로그램을 작성하시오.

# 입력
첫째 줄에 준하가 가입하려고 하는 사이트에 이미 존재하는 아이디가 주어진다. 아이디는 알파벳 소문자로만 이루어져 있으며, 길이는 50자를 넘지 않는다.


# 정답
```swift
let id = readLine()!

print("\(id)??!")
```
* readLine() 함수를 이용하여 준하가 가입하려고 하는 사이트에 이미 존재하는 아이디를 입력받는다.

* 그리고 print() 함수를 이용하여 준하의 놀람을 출력받는다.


* 주어진 조건에 따르면 이미 존재하는 아이디가 주어지고, 이 아이디는 알파벳 소문자로만 이루어져 있으며 길이가 50자를 넘지 않는다는 조건이 있다. 따라서 입력받은 아이디가 이러한 조건을 만족한다면 별도의 예외처리가 필요하지 않다.
