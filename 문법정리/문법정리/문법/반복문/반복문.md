# For in 
*  여러 데이터를 하나씩 꺼내면서 반복하는 문법
* 배열이나 범위를 하나씩 돌면서 실행하는 반복문 이라는 뜻
```swift
for 변수 in 데이터 {
    실행
}
```
```swift
let arr = [1, 2, 3]

for i in arr {
    print(i)
}
// 1,2,3,한개씩 출력
```

# While
* 조건이 참(true)인 동안 계속 반복하는 반복문
* 이 조건이 맞는 동안에 계속 실행해라 라는 뜻
```swift
while 조건 {
    실행
}
```
*사용 예시*
```swift
var i = 1

while i <= 5 {
    print(i)
    i += 1
}
// 5가 될때까지 계속 반복
```

**repeat-while**
* 무조건 한 번 실행하고 조건을 검사 함
```swift
repeat {
    실행
} while 조건
```
*사용 예시*
```swift
var i = 1

repeat {
    print(i)
    i += 1
} while i <= 5
// 1~5 출력
```

**while 과 repaet-while 차이점**
* While은 조건을 먼저 검사한다
* Repeat-while은 실행 후 조건을 검사함
