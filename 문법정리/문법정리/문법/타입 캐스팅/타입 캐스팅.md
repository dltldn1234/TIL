# 타입캐스팅
* 데이터(객체)의 타입을 확인하거나 바꾸는 것

**스위프트의 타입캐스팅은인스턴스의 타입을 확인하는 용도**

**또는 클래스의 인스턴스를부모 혹은 자식 클래스의 타입으로 사용할 수 있는지
확인하는 용도로 사용한다**

**is, as를 사용함**


## 타입 확인

is를 사용하여 타입을 확인한다
```swift
let num = 10

if num is Int {
    print("Int 타입")
}
// 결과 = int타입
```

## 업 캐스팅
* 자식 타입을 부모 타입으로 바꾸는 것(레벨 업 시켜준다고 생각하면 됨!)

as를 사용하여 부모클래스의 인스턴스로 사용할 수 있도록 컴파일러에게 타입정보를 전환해준다
```swift
class Animal {}
class Dog: Animal {}

let dog = Dog()
let animal: Animal = dog
// 업그레이드 
```

Any 혹은 AnyObject로도 타입 정보를 변환할 수있음!

암시적으로 처리되므로 생략해도 괜찮다



## 다운 캐스팅
* 부모 타입을 자식 타입으로 바꾸는 것

as? 또는 as! 를 사용하여

자식 클래스의 인스턴스로 사용할 수 있도록 컴파일러에게 인스턴스의 타입정보를 전환해줌
```swift
class Animal {}
class Dog: Animal {}

let animal: Animal = Dog()
let dog = animal as? Dog
// 다운그레이드
```

## 조건부 다운 캐스팅 
* 이 객체를 특정 타입으로 바꿔보되, 실패하면 안전하게 nil로 처리하는 

**as?**
```swift
class Animal {}
class Dog: Animal {}

let animal: Animal = Dog()

if let dog = animal as? Dog {
    print("Dog 타입")
}
// as?는 다운캐스팅이 성공하면 값을 반환하고, 실패하면 nil을 반환한다
```


## 강제 다운 캐스팅 
* 부모 타입을 자식 타입으로 “무조건” 바꾸는 것

**as!**

```swift
class Animal {}
class Dog: Animal {}

let animal: Animal = Dog()
let dog = animal as! Dog
// as!는 다운캐스팅을 강제로 수행하며, 실패 시 런타임 오류가 발생한다
```
