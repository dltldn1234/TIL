# Force Unwrapping 
**옵셔널의 값을 강제로 추출**

*예시*
```swift
func printName(_ name: String) {
    print(name)
}

var myName: String? = "yagom"

printName(myName!)  // yagom

myName = nil

print(myName!)
// 강제추출시 값이 없으므로 런타임 오류 발생

var yourName: String! = nil

printName(yourName)
// nil 값이 전달되기 때문에 런타임 오류 발생
```
