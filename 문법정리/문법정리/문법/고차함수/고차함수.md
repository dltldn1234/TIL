# 고차함수
* 함수를 “값처럼” 다루는 함수

전달인자로 함수를 전달받거나 함수샐행의 결과를 함수로 반환하는 함수

**map, filter, reduce**


## map (배열의 각 요소를 변환해서 새로운 배열을 만드는 함수)

* 컨테이너(데이터를 담아두는 그릇(자료 구조)) 내부의 기존 데이터를 변형하여 새로운 컨테이너 생성
```swift
let numbers: [Int] = [0, 1, 2, 3, 4]
var doubledNumbers: [Int]
var strings: [String]
```

## filter
* 조건에 맞는 값만 골라서 새로운 배열로 만드는 함수
```swift
var filtered: [Int] = [Int]()

for number in numbers {
    if number % 2 == 0 {
        filtered.append(number)
    }
}

print(filtered) // [0, 2, 4]
```


## reduce
* 배열의 모든 값을 하나로 합치는 함수
```swift
let someNumbers: [Int] = [2, 8, 15]

----------------------------------------------------
var result = 0

for number in someNumbers {
    result += number
}

print(result) // 25
```

## map / filter / reduce 차이

- map → 전부 변환
- filter → 조건 걸러냄
- reduce → 하나로 합침
