# 이름짓기

## Lower Camel Case
- 첫단어는 소문자로 표기
- 이후 단어들은 대문자로 표기
- 띄어쓰기나 언더바(_) 사용이 안되므로 이어쓰기로 표기

**사용예시**
- 함수(fuction)
- 매소드(method)
- 변수(variable)
- 상수(constant)
```swift
var userName = "siwoo"
```

## Upper Camel Case
- 모든 첫단어는 대문자로 표현 합니다
- 단어 사이에 띄어쓰기나(_)언더바 없이 붙여 씁니다

**사용예시**
- 클래스(class)
- 구조체(struct)
- 열거형(eunm)

```swift
class UserProfile {
    var name: String = "siwoo"
}
```
