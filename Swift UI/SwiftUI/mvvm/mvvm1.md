### 디코딩 이란 언어를 다른 언어로 바꾼다는것이다

* 요청과 응답 사이에는 디코딩이 있다

서버-> Json-> swift로 디코딩이 된다

### Json의 구성요소 : Key, Value로 분류가 된다 

* 서버에서 Json으로 보내면서 Key 랑 Value로 바껴서 swift로 들어간다

* DB에 저장된 값이 예를 들어서 제목: 시우, 설명: 물결이 대머리, 날짜: today라고 하면

Json으로 변환을 시키면 
```swift
제목=key  / 시우= str
설명=key  / 물결이 대머리=str
날짜=key  / today= int
```
이렇게 된다

model에서 하는일은 데이터의 형식만 정해주는 것 이다

필요한 요구사항을 model에서 정해주는 거
