# Swift Package는 Swift에서 모듈식으로 코드를 관리하고 배포할 수 있도록 제공하는 개념이다


**Swift Package는  라이브러리뿐만 아니라, 프레임워크, 실행가능한 모듈로서 분리, Swift 매크로 등 다양한 코드 묶음을 만들 수 있다**
* 쓰는 이유: 재사용되는 코드를 효율적(의존성 관리, 버전 관리 등)으로 활용할 수 있다는 점"이 가장 큰 이유가 된다


"Swift Package"들을 프로젝트에 추가, 관리하고 생성, 배포, 통합까지 하는데 사용할 수 있는 Apple의 공식 툴이 바로 'SPM (Swift Package Manger)'이다

예전에는 Cacapods 같은 것들을 많이 사용했지만 요새는 대부분의 작업을 spm에서 한다(apple이 spm을 지원해서 더욱 많이 쓰는듯히디)
